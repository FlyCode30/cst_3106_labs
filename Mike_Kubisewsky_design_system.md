 # Michael Kubisewsky's Portfolio Design System Documentation 

The following documentation provides an overview of the design system used for this professional portfolio project. The template for this portfolio was bought from a 3rd party, and so many of the design elements will come from that template. This will include layouts, components, typography, as well as screenshots, but there will also be some of my own design elements associated with content specific to my contents

## The Template

The template came from a website BootstrapMade. It was chosen for it's simple yet pleasing aesthetic and it's built in navigational features. Much of the existing template will be used, with the exception of the projects section, as it is not suitable for the kind of tech oriented projects that would be posted. The following is the link to the template source

[Template](https://bootstrapmade.com/personal-free-resume-bootstrap-template)

### 1. Color Palette

The overall color palette is a dark mode/black website that will use warm tones for accents. The structure for the template includes global colors, nav menu colors, and color presets that override the global colors. Since it's an overview, only the global colors will be included

**Global colors**

background-color: #000000
default-color: #fafafa (used for the majority of text)
heading-color: #F5E8D8
accent-color 1: #FF6F61
accent-color 2: #DAA520
surface-color: #1a1a1a
contrast-color:# 000000

### 2. Typography

The template uses several fonts, so this documentation will only include some of the default ones. Adjustments will be made later on

Default fonts: Roboto, Segoe UI, Helvetica Neue, Noto Sans
Heading fonts: Raleway
nav-font: Poppins

*** Conclusion

Overall, the aim of this design was to go with something that is minimal and easy to navigate. A dark theme was choosen because it is a more modern design due to it's easiness on the eyes, better battery life for mobile devices, and it's overall user experience.
