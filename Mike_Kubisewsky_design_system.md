 # Michael Kubisewsky's Portfolio Design System Documentation 

The following documentation provides an overview of the design system used for this professional portfolio project. The template for this portfolio was bought from a 3rd party, and so many of the design elements will come from that template. This will include layouts, components, typography, as well as screenshots, but there will also be some of my design elements associated with content specific to my contents

## The Template

The template came from a website BootstrapMade. It was chosen for it's simple yet pleasing aesthetic and it's built in navigational features. Much of the existing template will be used, with the exception of the projects section, as it is not suitable for the kind of tech oriented projects that would be posted.
